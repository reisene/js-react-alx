Kurs JS + React - zadania domowe

Poniższy projekt to zbiór zadań domowych realizowanych podczas kursu JavaScript + React organizowanego przez ALX.

67_1

Formularz z walidacją. Zadanie rozpoczęte na zajęciach, jako zadanie domowe było dodanie walidacji formularza przed wysłaniem.

51_1

Zadaniem było przypisanie wartości do 9 zmiennych i ułożenie ich w poprawne zdanie (z pomocą dodatkowej zmiennej).

15_2

Obliczanie silnii. Użytkownik podaje liczbę, a program za pomocą pętli oblicza silnię.

19_3

Użytkownik podaje serię liczb całkowitych, a podanie liczby 0 kończy serię (liczba 0 nie liczy się do statystyk). Program na podanych liczbach wykonuje obliczenia i wyświetla je.

59_2

Program zadaje 5 pytań o wynik mnożenia 2 losowych liczb (z zakresu 1-10). Przy podawaniu wyniku podaje informację o tym, czy odpowiedź jest prawidłowa czy błędna.

59_2a

Jak wyżej, program zadaje 5 pytań o wynik mnożenia 2 losowych liczb (z zakresu 1-10). Różnicą jest to, że przy podaniu błędnej odpowiedzi program ponownie zadaje to samo pytanie.

59_1

Program zadaje 10 pytań. Przy każdym pytaniu losuje 2 liczby i operator działania (dodawanie, odejmowanie lub mnożenie). Po ukończeniu quizu program pokazuje statystyki, na ile pytań użytkownik odpowiedział błędnie, a na ile poprawnie.

26_5a

Użytkownik podaje dowolną ilość liczb całkowitych, gdzie liczba 0 kończy jego działanie. Po zakończeniu program pokazuje liczbę wystąpień każdej z liczb podanych przez użytkownika.

22

Program oblicza sumę i średnią z 5 przekazanych liczb za pomocą funkcji.

27

Program, który do funkcji pobiera 3 zmienne - dwie pierwsze to liczby, trzecia to rodzaj wykonywanego działania. Na podstawie 3 parametru, program wyświetla sumę, różnicę, iloraz, iloczyn lub resztę z dzielenia.

67_1(2)

Moja własna wersja zadania 67_1, gdzie rozszerzyłam walidację o dodatkowe komunikaty w zależności od tego, czego użytkownik nie wprowadził. Dodatkowo, gdy użytkownik wprowadzi wszystkie wymagane dane, hasło wyświetlane w części podsumowania jest maskowane.
